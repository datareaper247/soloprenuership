# Ops Automation Engine — The Leverage Multiplier for Solo Founders

## Auto-Trigger (No Slash Command Needed)

Fires automatically when:
- "I keep doing this manually" / "repetitive task" → automation protocol
- "I want to hire a VA" / "need help with X" → document-first, then delegate
- "I spend too much time on [X]" → time audit + automation triage
- "onboarding is manual" / "every new customer requires" → customer onboarding automation
- "I'm the bottleneck" → founder extraction protocol
- "how do I systematize" / "build SOPs" → SOP builder
- "overwhelmed" / "can't keep up" → capacity audit + leverage plan

---

## Why This Exists

Solo founders die from two causes: wrong strategy and wrong allocation of time.

The wrong strategy problem is solved by the other skills. This skill solves the time problem.

At $0-10K MRR: **you should be in every customer interaction** — learning, selling, retaining.
At $10K+ MRR: **you should be extracting yourself from every repeatable process** — building leverage.

The founders who reach $1M ARR solo don't work harder — they build systems that work without them.

**Chanakya's Amatya principle**: "A king who does everything himself is not a king — he is a servant. A king who has excellent ministers is invincible."

Your systems are your Amatya (ministers). Every process you document and automate is a minister you never have to pay.

## THE TIME AUDIT PROTOCOL

### Step 1: The Activity Log (Fire Before Any Automation Decision)

```
TIME AUDIT FRAMEWORK:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Track every activity for 5 business days:
→ Activity name
→ Time spent (minutes)
→ Frequency (daily/weekly/monthly)
→ Category: CUSTOMER / PRODUCT / REVENUE / ADMIN / LEARNING
→ Value question: "Does this directly create revenue or retain customers?"

CALCULATE:
→ Monthly hours per activity = (minutes/session) × (sessions/month) / 60
→ Annual hours per activity = monthly × 12

SORT BY: Annual hours, descending.
TOP 3 time consumers = your automation/delegation targets.
```

### Step 2: The 4-Box Automation Triage

```
                │ HIGH LEVERAGE    │ LOW LEVERAGE
────────────────┼──────────────────┼──────────────────
REPEATABLE      │ AUTOMATE FIRST   │ DELEGATE OR KILL
                │ (high ROI)       │ (eliminate if possible)
────────────────┼──────────────────┼──────────────────
NOT REPEATABLE  │ DO IT YOURSELF   │ KILL IT
                │ (requires you)   │ (wastes everyone's time)
```

**Automate First (top priority)**:
- Customer onboarding sequences
- Invoice/payment reminders
- Support ticket triage
- Social media scheduling
- Weekly metrics reports
- Renewal/upsell emails

**Delegate (after documenting)**:
- Research tasks (VA)
- Content editing (editor)
- Customer support Tier 1 (CS VA)
- Bookkeeping (fractional CFO)

**Kill immediately**:
- Meetings that could be async messages
- Reports nobody reads
- Manual processes with <5 monthly occurrences AND low stakes

## THE CUSTOMER ONBOARDING AUTOMATION STACK

### The "White Glove to Automated" Transition

```
STAGE 1 ($0-5K MRR): MANUAL WHITE GLOVE
→ Personal welcome email within 2hrs
→ Direct Slack/WhatsApp invite
→ 30-minute onboarding call
→ Handwritten follow-up notes
→ Purpose: LEARN what customers need to activate

STAGE 2 ($5-20K MRR): SYSTEMATIZED PERSONAL
→ Templated welcome email (but still from your email)
→ Self-serve onboarding checklist (Notion/Coda template)
→ 15-minute "fast start" video (recorded once, used forever)
→ 3-email activation sequence (Days 1, 3, 7)
→ Purpose: REPLICATE what worked in Stage 1

STAGE 3 ($20K+ MRR): AUTOMATED WITH ESCALATION
→ Automated email sequence (ConvertKit/Customer.io)
→ In-app onboarding checklist (product feature, not manual)
→ Health score (usage events) → auto-flag low-health accounts
→ Human touch only for: enterprise, high ACV, at-risk signals
→ Purpose: SCALE without linear headcount growth
```

### The 7-Email Activation Sequence Template

```
Day 0 (signup): Welcome — "Here's the ONE thing to do first"
Day 1: Quick win — "Did you do [first action]? Here's why it matters"
Day 3: Social proof — "What [customer type like you] achieved"
Day 7: Feature depth — "Most people miss this. Here's the #1 advanced feature"
Day 14: Check-in — "How's it going? [Personal question based on ICP]"
Day 21: Upgrade prompt (if usage qualifies)
Day 30: NPS survey — Sean Ellis + "What almost made you not sign up?"
```

## THE FOUNDER EXTRACTION FRAMEWORK

### Removing Yourself from Critical Paths

```
EXTRACTION AUDIT (fire when: "I'm the bottleneck" detected):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

For each critical process, answer:
1. Can this happen without me? [Yes / No / With documentation]
2. If I'm unavailable 2 weeks — what breaks? → START THERE
3. What's the single highest-value thing only I can do? → PROTECT THAT TIME

EXTRACTION SEQUENCE:
Step 1: Document the process (SOP template above)
Step 2: Record a Loom video doing it once
Step 3: Have someone else follow the SOP → capture gaps
Step 4: Delegate with weekly review for 4 weeks
Step 5: Hand off completely with monthly check-in

What should NEVER be extracted:
→ Strategic decisions (reversibility <5/10)
→ Key customer relationships (top 20% of revenue)
→ Product vision and roadmap decisions
→ Hiring decisions

What MUST be extracted at $20K+ MRR:
→ Tier 1 customer support
→ Social media scheduling
→ Invoice and collections
→ Basic bookkeeping
→ Repetitive outreach
```

## THE DELEGATION LADDER

### The Right Order to Delegate

```
DELEGATION SEQUENCE (in order — don't skip levels):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Level 1: AUTOMATE (no human needed)
→ Scheduled emails, payment reminders, form→database, report generation

Level 2: TOOL (software replaces your manual work)
→ Calendly replaces scheduling, ProfitWell replaces spreadsheet tracking

Level 3: TEMPLATE (you create once, used repeatedly)
→ Email templates, proposal templates, onboarding docs, demo scripts

Level 4: AI (Claude/GPT as your always-available junior employee)
→ First drafts, research, summarization, classification, pattern-finding

Level 5: FRACTIONAL/ASYNC VA (human for non-automatable work)
→ Research, outreach personalization, Tier 1 support, social media posting
→ Start with: Fiverr, Upwork, OnlineJobs.ph (for async-first hires)
→ Duration: project-based first, retainer after proven fit

Level 6: FRACTIONAL EXPERT (specialized skill, not full-time)
→ Part-time CFO ($500-2000/mo), part-time CMO, part-time CTO
→ At $20K+ MRR for CFO, $30K+ for CMO, $50K+ for CTO equivalent

Level 7: FULL-TIME HIRE (only when Level 6 can't keep up)
→ Reversibility: 3/10. Only after 90-day fractional trial.

Chanakya rule: "A minister who knows less than the king weakens him.
A minister who knows more in their domain strengthens him."
Only hire when the delegate knows more about the function than you do.
```

→ Single focus: build the ONE thing that matters most

→ Follow up on all open proposals
→ Customer calls and check-ins
→ Outreach to new prospects (30 DMs)

FRIDAY (learning + admin — operations hat):
→ Review week against plan
→ Write build-in-public post
→ Handle admin (invoices, bookkeeping)

Review weekly: which 3 are the 3? Kill or park the rest.
