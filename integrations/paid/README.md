## Recommended Stacks by Stage

### Bootstrapped ($0-$500 MRR)
Everything open source. See [open-source/README.md](../open-source/README.md).

### Early Traction ($500-$5K MRR)
- **Research**: Perplexity API ($20)
- **SEO**: Ahrefs ($99) or Semrush ($119)
- **Sales**: Apollo.io free tier + Instantly ($37)
- **Email**: Resend ($20) + ConvertKit ($29)
- **Analytics**: PostHog free tier
- **Payments**: Stripe (usage-based)
- **Total**: ~$200-300/month

### Growth ($5K-$50K MRR)
Add: HubSpot ($45), Mixpanel ($28), Datadog ($30), Clay ($149), Hotjar ($39)
**Total**: ~$600-800/month

### Scale ($50K+ MRR)
Add: Segment ($120), Amplitude ($61), Vanta/Drata ($500), Datadog full stack
**Total**: ~$1,500-3,000/month
