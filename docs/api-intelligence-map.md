*When to unlock*: First paying customer acquired, need repeatable outbound pipeline.

| Tool | Free Tier | Cost | MRR Gate | Unlocks | Best For |

## API Stack Decision Matrix

Use this when deciding which APIs to pay for:

```
1. Is the problem validated? (3+ customers described the pain)
   No → don't buy a tool to solve it

2. Is the manual version taking >5hrs/week?
   No → keep doing it manually

3. Is the tool replacing a hire?
   If yes → tool almost always wins (10x cheaper)

4. Does the ROI math work?
   Tool cost < (hours saved × your hourly rate) within 30 days
   If no → not yet
```
