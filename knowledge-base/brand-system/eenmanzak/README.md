## Core recommendation

Start with a **3-layer identity model**:

1. **Founder identity** — conviction, expertise, category design, build journey, partnerships, Product Hunt launches.
2. **Company identity** — official trust shell, proof, hiring, partnerships, legal/contact/media credibility, central traffic router.
3. **Small number of product/app identities** — only when a product has its own audience, support surface, roadmap, or launch motion.

Default to **monobrand first**. Add product brands only when they earn their own distribution engine.
