# HS Code Classification Software — REJECTED

## Why Not
- 20+ competitors (Avalara, Zonos, KYG, Descartes, Digicust, TariffPilot, etc.)
- All use AI — no differentiation possible
- SMB importers prefer customs brokers (1-3% of shipment value) over software
- Software only cost-effective at 100+ orders/month
- Consolidation accelerating — small players getting acquired or killed
- Barriers to entry high: regulatory databases + legal expertise required

## Decision: AVOID
Market gaps exist (FTA optimization for SMBs, compliance monitoring) but too small vs. entrenched competitors.
